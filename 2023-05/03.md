# 分かったこと

- 不正な遷移があった場合ログイン画面にリダイレクトさせる方法

- `const value = inputValue ?? 42;`はinputValueがnullishの場合のみ42を代入できる
- コールバック関数とは関数の引数として渡される関数のこと
